- Okio：A modern I/O library for Android, Kotlin, and Java.
- 官网：https://github.com/square/okio
- Okio 基础使用部分：https://www.jianshu.com/p/3e0935bf2d45

- 特点：
  - 它也是基于插管的，而且是单向的，输入源叫 Source，输出目标叫 Sink
